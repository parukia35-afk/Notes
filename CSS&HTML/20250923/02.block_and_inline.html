區塊級元素

例子:div、h1~h6、p、ul、li、table、td、tr...

特性1:獨佔一整行，占滿整個父元素的寬度。即便設定寬度，下方的元素也不能使用它剩餘的空間(前、後換行)。

特性2:可以"正常設定"width,padding,margin,border等屬性。

特性3:不可被vertical-align屬性影響

行內級元素

例子:span,a,strong,em,b,i...

特性1:本身內容寬度多少，行內元素寬度就多少

特性2:inline元素之間會有空隙(幽靈空白)產生，這是行內元素的特性所致

特性3:會在文件流向由左至右、由上而下排列(當父元素容納不下時)。

特性4:無法設定width,height

特性5:可以被vertical-align屬性影響
